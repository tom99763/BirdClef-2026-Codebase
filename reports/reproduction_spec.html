1. Pipeline Overview

Phase A: Noisy Student (NS) — Independent Self-Training ============================================================= Step 0: Perch Teacher Perch v2 (frozen) + fine-tuned head → predictions on all 10,658 soundscapes → outputs/perch_teacher_aug_all_ss.csv Step 1: R0 Pseudo Labels perch_teacher_aug_all_ss.csv → gamma=2.0, threshold=p95 → pseudo_labels/ns_r0_perch_aug.csv Step 2: B0 NS Chain (R1→R12) R1→R4: Perch teacher blended (w decreasing 0.50→0.10) R5+: Pure self-training (perch_w=0.00) Each round: 5-fold train → infer_all_ss → Residual Corrector → gen_pseudo Step 3: PVT NS Chain (R1→R8) Seeded from B0 R4 pseudo labels (one-time) Then independent self-training R1→R8 Phase B: Noisy Classmate (NC) — Cross-Architecture Co-Evolution ===================================================================== Step 4: NC PVT R9+ Blend B0 R11 + PVT R8 predictions → NC pseudo labels Phase 2: Confidence-weighted blending Phase 3: Disagreement mining (sample weights) Phase 4: Soft distillation loss (R10+) Step 5: NC Bidirectional (R10+) PVT R10 predictions feed back to B0 R12+ Alternating: PVT round → B0 round → PVT round → ... Phase C: Ensemble & Submission ================================ Step 6: ONNX Export Best fold checkpoints → FP32 ONNX + INT8 quantized Step 7: Kaggle Notebook 3-model SED ensemble + Perch probe → VLOM blend → submission

3. Step 1 — R0 Pseudo Labels

python3 scripts/gen_pseudo_ns.py \
    --round 0 --clip_sec 20 \
    --perch_csv outputs/perch_teacher_aug_all_ss.csv \
    --perch_w 1.0 --sed_w 0.0 \
    --percentile 95 --gamma 2.0 \
    --nonaves_perch_only \
    --out pseudo_labels/ns_r0_perch_aug.csv
Key settings:
  • --clip_sec 20: Aligns pseudo labels to 20s training clips (row_ids: _20, _25, ..., _65)
  • --gamma 2.0: Power transform sharpens soft labels (BirdCLEF 2025 1st place technique)
  • --percentile 95: Per-class dynamic threshold at 95th percentile of positive predictions
  • --nonaves_perch_only: For non-Aves classes (Amphibia/Insecta/Mammalia/Reptilia), use only Perch predictions (SED unreliable for these taxa with ~12.9 clips/species)

Per-Round Pipeline

For each round R:
  1. Train 5 folds:
     python3 train_sed_ns.py --config configs/sed_ns_b0_20s_r{R}.yaml --fold {0..4}

  2. 5-fold ensemble inference on all soundscapes:
     python3 train_sed_ns.py --config ... --infer_all_ss
     → outputs/sed-ns-b0-20s-r{R}/all_ss_probs.npz

  3. Residual Corrector:

         --round {R} --alpha 0.40
     → outputs/sed-ns-b0-20s-r{R}/all_ss_probs_corrected.npz

  4. Generate pseudo labels (if R < max_round):
     python3 scripts/gen_pseudo_ns.py \
         --round {R} --sed_dir outputs/sed-ns-b0-20s-r{R} \
         --perch_w {PW} --sed_w {SW} --percentile {PCT} --gamma {G} \
         --nonaves_perch_only \
         --out pseudo_labels/sed_20s_r{R}.csv

Why a separate PVT chain? PVT (Vision Transformer) has fundamentally different inductive biases from EfficientNet-B0 (CNN): CNN excels at local spectral texture; ViT excels at global temporal context. Running an independent NS chain produces predictions with low correlation to B0 → high ensemble diversity → better LB.

Phase 2 Detail: Confidence-Aware Blending

# For each sample i:
conf_B0[i] = 1 / entropy(B0_probs[i])     # lower entropy = higher confidence
conf_PVT[i] = 1 / entropy(PVT_probs[i])

# Normalize (with base weights)
w_B0[i] = (0.5 * conf_B0[i]) / (0.5 * conf_B0[i] + 0.5 * conf_PVT[i])
w_PVT[i] = 1 - w_B0[i]

pseudo[i] = w_B0[i] * B0_probs[i] + w_PVT[i] * PVT_probs[i]

Phase 3 Detail: Disagreement Mining

# Per-sample disagreement score:
variance[i] = mean_over_species( Var(B0_probs[i], PVT_probs[i]) )

# Training weight:
_nc_weight[i] = 1.0 + alpha * (variance[i] / max_variance)
# alpha = 2.0: max-disagreement samples get 3x training weight
# Stored in _nc_weight column of pseudo CSV
# Read by PseudoSoundscapeDataset, applied in training loop

Phase 4 Detail: Soft Distillation Loss

# NCDistillLoss in train_sed_ns.py
class NCDistillLoss:
    L = (1 - beta) * FocalBCE(logits, hard_targets)
      + beta * BinaryKLD(sigmoid(logits/T), soft_targets) * T^2

    # beta = 0.3, T = 2.0
    # soft_targets = raw ensemble blend probabilities (pre-threshold)
    # hard_targets = thresholded binary labels

# Config (add to training YAML):
training:
  nc_distill_beta: 0.3
  nc_temperature: 2.0

Phase 5 Detail: Bidirectional Loop

# Automated in scripts/auto_nc_full.sh
# 6 generations of alternating co-evolution:

for GEN in 1 2 3 4 5 6:
    # PVT next round: blend B0_latest + PVT_latest → train PVT
    gen_nc_pseudo(b0=B0_LATEST, pvt=PVT_LATEST) → PVT R(N+1)
    train PVT R(N+1) → infer → corrector

    # B0 next round: blend PVT_latest + B0_latest → train B0  (KNOWLEDGE FLOWS BACK!)
    gen_nc_pseudo(pvt=PVT_LATEST, b0=B0_LATEST) → B0 R(M+1)
    train B0 R(M+1) → infer → corrector

VLOM Blend

# Variance-weighted log-odds mean
# SED and Perch predictions combined in log-odds space
def vlom_blend(sed, perch, w_sed=0.70, w_perch=0.30):
    eps = 1e-9
    lo_sed   = log(sed / (1 - sed + eps) + eps)
    lo_perch = log(perch / (1 - perch + eps) + eps)
    blended  = w_sed * lo_sed + w_perch * lo_perch
    return sigmoid(blended)
