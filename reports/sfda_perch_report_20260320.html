1 什麼是 Source-Free Domain Adaptation (SFDA)?

傳統 Domain Adaptation (DA) 的問題

標準 UDA (Unsupervised DA) 需要同時存取 source 和 target domain 的資料。但在實際應用中，source 資料往往因為：隱私法規限制、儲存成本、IP 問題、或只有 model checkpoint 而無法取得原始資料。

SFDA 的核心概念

SFDA 只使用 已訓練好的 source model + unlabeled target 資料，不需要任何 source domain 資料。目標是讓 model 自適應到 target distribution，同時保留 source 學到的知識。

關鍵定義： SFDA = 給定 source pre-trained model fs + unlabeled target data Xt，學習 adapted model ft，使得 ft 在 target distribution 上有更好的 performance，且不能使用任何 source domain data。

關鍵 Domain Gap 維度 (4種並存)：
1. 地理偏移 (Geographic Shift)
Source 覆蓋全球，target 只在巴西 Pantanal。南美洲的聲紋環境與北美/歐洲截然不同，背景音種類（樹蛙、昆蟲群落）完全不同。
2. 錄音設備偏移 (Device Shift)
指向性麥克風（XC 錄音師用）vs 全向性 ARU 設備。頻率響應、SNR、增益設定完全不同，導致同一物種的頻譜特徵有系統性偏差。
3. 標籤偏移 (Label Shift)
Source: 單一物種精準標注（class prior 均等）。Target: 多物種 co-occurrence（class prior 偏向 Pantanal 常見種），且很多 source 物種在 target 幾乎不出現。
4. 任務偏移 (Task Shift)
Source 是分類（哪一種鳥）。Target 是偵測（5秒內有沒有這種鳥），且需要同時偵測多個物種。Multi-label detection 比 single-label classification 難得多。
Domain Gap 的具體影響：同一個 SED model 在不同 validation set 的 AUC 差距
3 為什麼 BirdCLEF 2026 適合用 SFDA?

✓ 條件 1: 有 pre-trained source model

Perch 是 Google 用 XenoCanto 全量資料訓練的 bioacoustic foundation model（EfficientNet-B1, 1536-dim embeddings）。這就是一個強大的 source model，且我們只有模型權重，不再需要原始訓練資料。

✓ 條件 2: Target domain data 可取得

10,658 個 target domain soundscape 檔案（unlabeled）+ 1,478 個有標注的 target clips（train_soundscapes_labels）。這比一般 SFDA 情境更有利——我們有 labeled target data。

✓ 條件 3: Test 與 Train soundscape 同分布

Test soundscapes 確認來自相同 Pantanal 地區（BC2026_Test_*_S05_* 檔名格式），與 train_soundscapes 同一批 ARU 部署。這表示可以用 train_soundscapes 做 target adaptation，直接提升 test performance。

✓ 條件 4: Perch 的 embedding 特性

Perch embeddings（1536-dim）是在 bioacoustic-specific task 上訓練的，cross-domain transfer 能力極強（Nature 2023 論文驗證可以 zero-shot 遷移到蝙蝠、鯨魚等非訓練物種）。這讓 SFDA 的出發點比一般 ImageNet pretrained model 更好。

BirdCLEF 2026 的特殊優勢：Semi-supervised DA，不是純 SFDA
我們有 1,478 個 labeled target clips（train_soundscapes_labels），這讓我們可以用比純 SFDA 更強的 semi-supervised domain adaptation。本質上是：
→ Perch (source model) + labeled soundscape clips (supervised target) + unlabeled soundscape clips (unsupervised target) = 三層資訊全部利用
4 主要 SFDA 方法介紹
3 TENT — Test-time Entropy Minimization
ICLR 2021 Online BN params only
核心概念： 在 inference 時，不更新整個模型，只更新 Batch Normalization 的 affine 參數（γ, β）。用 entropy minimization 做目標函數：

L_TENT = E[H(f(x))] = -E[Σ_c p_c · log(p_c)]

原理： 更 confident 的預測 → 更低 entropy → 更接近正確。直接在 target batch 上最小化 entropy，逼迫模型做出確定性預測。
BirdCLEF 限制： 針對 multi-label 情境，naive entropy minimization 會 collapse（讓所有 class 預測趨向 0）。需要改用 per-class binary entropy，並限制更新步數。
Effort: Low Risk: Medium (collapse) Expected: +1~2pp OOF
2 SHOT — Source Hypothesis Transfer
ICML 2020 Feature-level Pseudo-label
核心概念： 凍結 classifier head（source hypothesis），只 adapt feature extractor。用兩個 loss 聯合優化：

1. Information Maximization (IM)：
L_IM = -H(p̄) + λ·E[H(p)]
H(p̄) = marginal prediction entropy（maximize diversity）
E[H(p)] = per-sample entropy（minimize uncertainty）

2. Pseudo-label Self-training： 對 target embeddings 做 clustering，高信心預測當作 pseudo label，重新訓練 feature extractor。

在 Perch 上的應用： 凍結 Perch backbone，只 adapt 我們的 MLP probe head，用 unlabeled soundscapes 的 IM loss fine-tune。
Effort: Medium Risk: Medium Expected: +2~3pp OOF
1 LaplacianShot — Transductive Inference Already Implemented
ICML 2020 Transductive kNN graph
核心概念： 不在訓練時 adapt，而是在 inference 時對整個 test batch 做聯合最佳化。利用 embedding space 的局部結構：相鄰的 embeddings 應該有相似的預測。

Energy function：
E = Σ_i H(p_i) + λ · Σ_{i,j∈kNN} ||p_i - p_j||²

第一項：每個 clip 的預測要 confident（低 entropy）
第二項：kNN 鄰居的預測要一致（local smoothness）

BirdCLEF 優勢： Pantanal soundscape 中，同一地點同一時間段的聲音高度相關（同一鳥類在旁邊）。kNN graph 可以利用這種 co-occurrence 結構。
Effort: Already done Risk: Low Expected: +1~3pp OOF
4 NRC — Neighborhood Reciprocal Clustering
NeurIPS 2021 Graph-based Reciprocal kNN
核心概念： 對 kNN graph 中的邊做加權：reciprocal neighbors（A 在 B 的 k-NN，且 B 在 A 的 k-NN）給予更高 affinity。互相認識的鄰居比單向鄰居更可靠。

Self-regularization loss：使模型預測與 reciprocal neighborhood 的 consensus 一致，減少 noisy pseudo labels 的影響。

在 Perch 上的應用： 對所有 soundscape clips 的 Perch embeddings 建 kNN graph，用 reciprocal consensus 精鍊 probe 的預測。
Effort: High Risk: Medium Expected: +1~2pp OOF
1 FeCAM — Mahalanobis NCM Already Implemented
NeurIPS 2023 Prototype-based Covariance-aware
核心概念： 用 Mahalanobis distance 取代 Euclidean distance 做 nearest-class mean (NCM) classification。對每個 class 估計一個 covariance matrix（Ledoit-Wolf + Tukey regularization）。

為什麼重要： 不同鳥類的 embedding cluster 形狀差異很大（有些是長形橢圓，有些是球形）。Euclidean prototype 假設 spherical clusters，會有系統性誤差。

Domain 優勢： Covariance 可以分別在 train_audio 和 soundscape 上計算，做 domain-aware distance。
Effort: Already done Risk: Low Expected: +2~4pp OOF

新增步驟 1：ss_repeat=5
在 MLP 訓練時，target domain soundscape clips 重複 5 次。效果：739 clips × 5 = 3,695 items，佔總訓練資料 ~4.2%（原本 0.86%）。讓 MLP 的梯度更多來自 target distribution。
新增步驟 2：LaplacianShot inference
在 inference 時，對整個 soundscape batch 建 kNN graph（k=7），用 Laplacian regularization 讓鄰近 clips 的預測一致。不需要任何訓練，只在 inference 時計算。

Domain-Aware PCA 的數學直覺

Source-only PCA 的問題：PCA basis vectors 在 source distribution 的方向上找 maximum variance。當 target distribution 偏移時，重要的 target 方向可能在 source PCA 的低方差成分中，被 truncation 丟掉。

聯合 PCA 的修正：在 train+soundscape 上一起 fit，使得 PCA basis 同時捕捉兩個 domain 的高方差方向，target-specific 的特徵不會被截斷。

這與 DA 文獻中的 feature space alignment 概念一致（Coral, CAN），但用的是最簡單的 PCA 版本。
