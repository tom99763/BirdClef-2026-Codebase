2. Mel Spectrogram Frontend ⭐ 最大差異

參數 Competitor 我們 影響評估
Frontend type Learnable PCEN AmplitudeToDB (fixed log) 顯著 PCEN 自適應增益正規化
n_fft 1024 2048 中等 更細的頻率解析度
hop_length 512 512 相同
n_mels 128 224 中等 我們更細的 mel bins
fmin / fmax 40 Hz / 14,000 Hz 0 Hz / 16,000 Hz 小
Normalization PCEN (learnable 4 params/bin) Per-sample min-max 顯著
Input shape (1, 128, 313) (1, 224, 313) 中等
PCEN 詳情：公式 (S / (eps + M)^α + δ)^r - δ^r — 其中 M 是用 depthwise causal 1D conv 估計的背景雜訊。4 個 learnable params（α, δ, r, s）每個 mel bin 各自學習。 相比 fixed log compression，PCEN 能動態壓制背景噪音，對 train_audio OOF AUC 幫助大，但對 soundscape domain shift 幫助不確定。 我們目前有實作 PCEN (src/model/pcen.py)，但在 v30 系列沒有啟用。
